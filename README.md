# Projeto de Validações com Regex

### Curso Realizado

![certificate](https://user-images.githubusercontent.com/88117244/198894255-d4ae8e68-8650-48c1-9cb8-184cd9c87732.png)

Este projeto utiliza expressões regulares para validar diferentes tipos de campos e é destinado principalmente para fins de testes. Embora seja um projeto experimental, foi desenvolvido com o objetivo de aprender e praticar o uso de regex em projetos reais.

### Aprendizado de Regex

Para aprimorar os conhecimentos em Regex, foram utilizados os seguintes recursos:

- [Masterclass Flutterando](https://masterclass.flutterando.com.br/public/products/827e7911-30fa-4ea3-9ec7-3a65183ee475)
- [Artigo sobre Regex na CWI Software](https://medium.com/cwi-software/e-o-regex-como-vai-657f94388dc)
- [Curso de Expressões Regulares da Alura](https://www.alura.com.br/conteudo/expressoes-regulares)

### Site Útil de Regex

Além disso, foi utilizado o site Regexr para auxiliar no desenvolvimento e teste das expressões regulares.

- [Regexr](https://regexr.com/)

## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- Flutter 3.7.6
- Dart 2.19.3

## Como Executar o Projeto

Para executar o projeto em sua máquina local, siga os seguintes passos:

1. Clone o repositório em sua máquina local.
2. Abra o projeto em seu editor de código preferido.
3. Execute o projeto usando o SDK do Flutter.
